## Contributing to Security

We welcome security research on Sovereign Communications. Here's how you can help:

### Responsible Disclosure
1. **Report Privately**: Use GitHub Security Advisories or email
2. **Allow Time**: Give us 90 days to fix before public disclosure
3. **Provide Details**: Include steps to reproduce
4. **Coordinate**: Work with us on disclosure timing

### In Scope
- Web application
- Android application
- iOS application
- Core protocol
- Cryptographic implementations
- Mesh networking layer

### Out of Scope
- Social engineering
- Physical attacks
- DoS/DDoS
- Third-party libraries (report to maintainer)

### Guidelines
- Don't access or modify user data
- Don't perform actions that could harm users
- Act in good faith
- Follow disclosure timeline

See [SECURITY.md](../SECURITY.md) for complete guidelines.
